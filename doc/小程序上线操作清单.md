## 上线前必须补齐

### 1. 本地环境变量

复制并确认根目录 `.env`：

```bash
cp .env.example .env
```

### 2. 上传音频到 CDN

本地音频目录：

```text
generated/audio
```

上传后必须保持目录结构不变，例如：

```text
https://cdn.example.com/generated/audio/shj/required-1/unit-1/digital/uk.mp3
https://cdn.example.com/generated/audio/shj/required-1/unit-1/digital/us.mp3
https://cdn.example.com/generated/audio/shj/required-1/unit-1/digital/zh.mp3
```

发布构建（推荐，会自动跑词库、测试、CDN 校验、构建）：

```bash
pnpm release:weapp
```

仅构建小程序：

```bash
pnpm build:weapp
```

单独校验 CDN：

```bash
pnpm audio:verify-cdn
```

### 3. 微信后台配置域名

在微信小程序后台配置 CDN 域名：

```text
小程序后台 -> 开发 -> 开发设置 -> 服务器域名
```

至少需要配置音频 CDN 所在 HTTPS 域名。域名需要已备案，不能使用 IP 或 localhost。

### 4. 微信开发者工具导入

构建后导入目录：

```text
dist/build/mp-weixin
```

导入后重点检查：

- 原生 tabBar 是否固定显示。
- 首页、生词本、听写三个 tab 是否切换正常。
- 进入体检、听写、报告、单词详情时 tabBar 是否隐藏。
- 全部单词 -> 单词详情：释义、例句、翻译、进度与发音。
- iPhone 和 Android 真机音频是否连续播放。
- 弱网或音频缺失时是否有合理提示。

## 仍需产品侧确认

- 正式上线词库是否必须包含“沪外教版”。当前工程词库只有沪教版。
- 是否首发就做用户账号和云同步。当前学习数据存在本机 `uniStorage`，不跨设备。
- 是否接入付费。若接入付费，需要补微信支付、订单、权益和用户体系。
- 是否采集学习数据到云端。若采集，需要补隐私指引、用户授权和后端数据库。
